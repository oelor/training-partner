# 🏆 Training Partner - Combat Sports Platform

A modern, sleek two-sided marketplace connecting combat sports athletes with training partners and partner gyms.

![Build Status](https://img.shields.io/badge/build-passing-green)
![Next.js](https://img.shields.io/badge/Next.js-14-black)
![React](https://img.shields.io/badge/React-18-blue)

## 🚀 Quick Start

### Run Locally
```bash
cd training-partner
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Deploy to Vercel (Free)
1. Push to GitHub
2. Import project in Vercel
3. Deploy!

See [SETUP.md](./SETUP.md) for detailed instructions.

## 📱 Features

- **Landing Page** - Modern dark theme with hero, features, pricing
- **Authentication** - Sign up/sign in with email
- **User Profiles** - Complete combat sports profile with sports, skill level, weight class, training goals, availability
- **Partner Matching** - Search & filter partners with match scores
- **Gym Listings** - Partner gyms with open mat hours (premium-locked)
- **Settings** - Account, notifications, subscription, privacy
- **Legal** - Terms of Service & Privacy Policy

## 🎨 Design

- Dark mode aesthetic
- Orange (#FF4D00) primary / Green (#00FF88) accent
